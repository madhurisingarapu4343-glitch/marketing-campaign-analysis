# Marketing Campaign Analysis

## 📌 Project Overview
This project analyzes customer data from marketing campaigns to identify high-value customers and improve campaign performance.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI

## 📊 Key Features
- Data Cleaning & Feature Engineering
- Customer Segmentation
- KPI Analysis
- Interactive Dashboard

## 📈 Insights
- High-income customers show higher response rates
- High spenders are the best target group
- Customers without children respond more to campaigns

## 📂 Files Included
- Dataset (CSV)
- Python Notebook
- SQL Queries
- Power BI Dashboard
- Project Report
